# Meccha Chameleon

Unofficial Meccha Chameleon guide site.

- Framework: Next.js 16 App Router, static export
- Hosting: Cloudflare Workers Assets
- Design source: `../design/stitch-r4-v2/`

## Commands

```bash
npm run lint
npm run build
npm run cf:dry-run
npm run deploy:workers
node scripts/verify-live.mjs
```
